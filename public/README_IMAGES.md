# Инструкция по добавлению изображений

## Логотип

Поместите файл логотипа в папку `public/` с именем `logo.png`

Логотип должен быть:
- Формат: PNG (с прозрачным фоном предпочтительно)
- Размер: минимум 200x200px (будет автоматически масштабироваться)
- Стиль: золотой Пегас с "1990" (как на прикрепленном изображении)

## Галерея

Поместите изображения маникюра в папку `public/gallery/` со следующими именами:

1. `gothic-halloween-nails.jpg` - Готический/Хэллоуин дизайн
2. `chrome-sunburst-nails.jpg` - Хром с солнечными лучами
3. `strawberry-nails.jpg` - Клубничная тема
4. `plaid-nails.jpg` - Клетчатый узор
5. `zebra-french-tip.jpg` - Зебра французский маникюр
6. `yellow-ombre-nails.jpg` - Желтый омбре
7. `neon-smiley-nails.jpg` - Неоновый смайлик
8. `grey-heart-nails.jpg` - Серый с сердечками
9. `blue-star-nails.jpg` - Синий со звездами

### Требования к изображениям галереи:
- Формат: JPG или PNG
- Размер: минимум 800x800px (квадратные изображения предпочтительны)
- Качество: высокое разрешение для четкого отображения

После добавления изображений они автоматически появятся на сайте.
